# 🛑 失败案例复盘：Concord (星鸣特攻)

## 1. 项目概况
*   **开发商**: Firewalk Studios (Sony)
*   **类型**: Hero Shooter (5v5)
*   **结局**: 上线 2 周后关服，全额退款，工作室解散。

## 2. 核心败因分析

### 2.1 市场迟钝 (Market Saturation)
*   **问题**: 游戏立项于 2016 年左右，那是 Overwatch 的巅峰期。但当 2024 年发售时，Hero Shooter 市场早已饱和 (OW2, Valorant, Apex)。
*   **后果**: 玩家没有理由离开他们已经投入了数千小时的游戏来玩一个收费 $40 的新游戏。

### 2.2 角色设计平庸 (Character Design)
*   **问题**: 角色缺乏“剪影辨识度”和美学吸引力。设计过于追求“写实”和“多元化”，导致角色看起来像普通的 Cosplayer，缺乏英雄气概 (Power Fantasy)。
*   **后果**: 无法吸引“为爱付费”的玩家群体，二创社区一片死寂。

### 2.3 缺乏核心识别度 (Identity Crisis)
*   **问题**: 玩法缝合了 Destiny 的手感和 Overwatch 的技能，但没有独特的创新点 (USP)。
*   **后果**: 被称为“平庸的极致”。

## 3. 对 Vampirefall 的启示

### ✅ 必须做的
1.  **剪影测试**: 每一个塔、每一个怪物，都必须在全黑剪影下能被一眼识别。
2.  **差异化**: 我们的“塔防 + 肉鸽 + 刷宝”混合品类本身就是护城河，必须强化这种混合体验，而不是做成“另一个 Kingdom Rush”。

### ❌ 绝对禁止
1.  **盲目追风**: 不要因为某个品类现在火就去立项。要看 3 年后市场缺什么。
