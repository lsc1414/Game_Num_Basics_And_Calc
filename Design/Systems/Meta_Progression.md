# 🌲 局外成长系统 (Meta-Progression)

Project Vampirefall 是 Roguelite，意味着玩家死亡后虽然装备丢失，但会保留部分资源用于永久强化。本系统旨在提供长期的目标感。

---

## 1. 资源循环 (Economy Loop)

*   **金币 (Gold):** *局内资源*。用于商店购买、刷新Perk。死亡清零。
*   **魂石 (Soul Shards):** *局外资源*。击杀 Boss 或完成成就获得。用于解锁天赋树。
*   **蓝图 (Blueprints):** *永久解锁*。解锁新的塔种类或武器制作配方。

## 2. 虚空星盘 (The Void Constellation) - *天赋树*

类似于《恐怖黎明》的星座或《流放之路》的天赋盘，但简化为局外解锁。

### 2.1 结构
*   **起始点:** 基地中心。
*   **分支:** 三大星系，对应三种玩法风格。
    1.  **战争星系 (War):** 强化角色本体属性 (攻击、血量)。
    2.  **工程星系 (Engineering):** 强化塔防机制 (造价降低、塔耐久)。
    3.  **财富星系 (Fortune):** 强化经济与掉落 (金币获取、MF值)。

### 2.2 节点类型
*   **小星点 (Minor Node):** +2% 攻击力。消耗少。
*   **大星点 (Major Node):** +10% 攻击力。消耗中。
*   **核心星点 (Keystone):** 改变机制。
    *   *例:* **[自动装填]:** 角色每 5秒 自动维修周围受损的塔。

## 3. 基地建设 (Base Building)

玩家拥有一个可升级的“避难所”。

*   **铁匠铺 (Blacksmith):**
    *   *功能:* 解锁新武器的锻造权限。
    *   *升级:* 允许锻造更高 Tier 的初始装备。
*   **炼金台 (Alchemist):**
    *   *功能:* 升级回复药瓶的容量和充能速度。
*   **档案馆 (Archive):**
    *   *功能:* 查看怪物图鉴、统计数据。

## 4. 赛季制/转生 (Seasons / Prestige)

为了解决大后期数值膨胀问题，引入类似“赛季”或“转生”的概念。

*   当玩家点满星盘后，可以开启 **[多重宇宙难度]**。
*   **代价:** 重置所有星盘等级。
*   **奖励:** 获得永久的 **[宇宙碎片]**，用于购买外观 (Skin) 或开启特殊的“挑战模式”词缀。

---

## 5. 设计陷阱规避 (Pitfalls)

*   **拒绝数值碾压:** 局外成长主要提供**“广度”** (更多的选择、更方便的功能)，而非单纯的**“强度”**。
    *   *坏设计:* +1000% 伤害。导致新手打不过的怪，老手闭眼打。
    *   *好设计:* 解锁“双持武器”能力。让老手能玩出新花样。
